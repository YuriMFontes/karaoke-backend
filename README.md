
# KaraokedoMaclim - Backend


## 🚀 Sobre o Projeto
KaraokedoMaclim é um sistema backend desenvolvido em Java para um aplicativo de karaokê offline. O projeto nasceu da ideia de ajudar meu pai, que ama cantar, proporcionando uma experiência fluida e acessível para reproduzir vídeos de karaokê armazenados localmente.


## Tecnologias Utilizadas

**Linguagem:** Java

**Framework:** Spring Boot

**Arquitetura:** MVC

**Gerenciamento de Dependências:** Maven
