# 🎤 KaraokedoMaclim

Projeto de Karaokê offline desenvolvido com carinho para meu pai, que ama cantar.  
Feito com **Java** e **JavaFX**, o sistema permite buscar músicas por número e reproduzir vídeos com letras sincronizadas, diretamente de um HD externo.

## ✨ Funcionalidades

- Interface simples e intuitiva com JavaFX
- Entrada de número da música na tela inicial
- Busca automática da música no HD externo
- Exibição do nome da música e cantor
- Reprodução offline do vídeo da música
- Letra sincronizada com o som durante a reprodução
- Compatível com computadores mais simples (baixo consumo de recursos)

## 💻 Tecnologias Utilizadas

- Java 11+
- JavaFX
- Padrão de projeto MVC
- Leitura de dados via arquivos JSON
- Manipulação de arquivos locais (HD externo)


## 🚀 Como Funciona

1. Ao abrir o programa, o usuário vê uma tela com um campo para digitar o número da música.
2. O número digitado é usado para buscar os dados da música no arquivo `musica.json`.
3. Se a música for encontrada, exibe o nome e o cantor.
4. Em seguida, o programa muda para a tela de vídeo.
5. O vídeo é carregado diretamente do HD externo e começa automaticamente.
6. As letras são exibidas em sincronia com o áudio da música.

## 🔌 Requisitos

- Java instalado (versão 11 ou superior)
- HD externo com os vídeos organizados por número
- Arquivo `musica.json` com os metadados das músicas
